Candy v Fruit
TIMEOUT (SEC)

Multiplayer network candy v fruit battleship game

  1. Click on the candy and fruit to move it to the grid
  2. When the game starts you will get a turn to choose a cell
  3. If there is candy on that grid point you will lose points
  4. If there if fruit you will gain points
  5. If there is a CoderDojo logo on its surpise

Enjoy!
